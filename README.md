# CardCutter

Clip audio from YouTube or Bilibili and capture subtitles (CC) as the clip filename for SuperMemo import.

## Features
- Capture current on-screen CC or collect subtitles within a time range.
- Record a video segment and export as MP3 or M4A.
- Trim the recorded audio before export.
- Use subtitles as the exported filename for easier SuperMemo import.

## How to Use
1. Open a YouTube or Bilibili video and enable subtitles/CC.
2. Click the CardCutter extension to show the panel.
3. Set Start and End using the buttons while the video plays, or type times manually.
4. Click "Use Current CC" to fill the subtitle box.
5. Click "Record" to capture the segment.
6. In the trim panel, adjust the start/end sliders if needed.
7. Click "Export" to download the audio file. The filename will be the subtitle text.

## Tips
- If no subtitles are found, make sure CC/字幕 is enabled in the player.
- For precise clips, use the nudge buttons to adjust Start/End by 0.1s.
- You can edit the subtitle text before exporting to control the filename.

# CardCutter（中文）

从 YouTube 或 Bilibili 截取音频，并把字幕（CC）作为文件名，方便导入 SuperMemo。

## 功能
- 读取当前屏幕上的字幕，或在时间范围内收集字幕。
- 录制视频片段并导出为 MP3 或 M4A。
- 导出前可在修剪面板中剪裁音频。
- 将字幕作为导出文件名，便于 SuperMemo 导入。

## 使用方法
1. 打开 YouTube 或 Bilibili 视频，并启用字幕/CC。
2. 点击 CardCutter 扩展打开面板。
3. 播放视频时点击 Start/End 按钮设置时间，或手动输入时间。
4. 点击“Use Current CC”填充字幕文本框。
5. 点击“Record”录制该片段。
6. 在修剪面板中调整起止时间（如有需要）。
7. 点击“Export”下载音频文件，文件名即字幕文本。

## 提示
- 如果提示没有字幕，请确认播放器已开启字幕/CC。
- 需要精确时间时，用 nudge 按钮以 0.1 秒调整起止点。
- 导出前可以编辑字幕文本以控制文件名。
